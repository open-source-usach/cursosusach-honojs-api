# CursosUSACH Hono.js API

Work in progress...

## Technologies

- TypeScript with Hono.js
- Cloudflare Workers, KV (nosql)
- Zod

## Getting Started

Development mode, run locally pressing L on your terminal

```bash
npm install
npm run dev
```

Deployment

```
npm run deploy
```
